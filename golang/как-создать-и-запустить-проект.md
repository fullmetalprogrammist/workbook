# Создать проект

- Создать папку, перейти в нее
- Консоль: `go mod init foobar`
- В папке появится `go.mod` файл





# Установка пакетов

## Установка пачкой

Если импортировать в программе какие-то пакеты, не относящиеся к стандартной библиотеке

```go
import (
	"github.com/jackc/pgx/v5/pgxpool"
	"github.com/joho/godotenv"
)
```

то можно командой `go mod tidy` установить все эти пакеты
