- supabase предоставляет библиотеку `supabase-js` для работы с собой.
  - ласт раз описание как поставить и как пользоваться было тут
    - https://supabase.com/docs/reference/javascript/introduction
- для работы библиотеки нужен апи ключ и url проекта
  - и то, и другое можно получить в настройках проекта
    - заходим на главную страницу своего аккаунта, там ищем раздел с проектами
    - заходим в нужный проект, ищем Project Settings, там API Keys, Data API и прочее, короче всю инфу, нужную для коннекта, там надо искать
