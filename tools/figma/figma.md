# Флоу

- Добавляем фрейм
  - Включаем в нем сетку
- Накидываем элементы, выравниваем по сетке

## Советы, заметки, замечания

- Если, допустим, страница очень длинная (лендинг), но делится как бы на отдельные блоки, можно каждый блок делать в отдельном фрейме, а потом просто фреймы в упор друг к другу придвигать. Визуально это все равно что сделать в одном фрейме, а технически может быть удобнее, например, выставлять отступы.
- Копировать \ вставлять можно много чего. Например, сетку в фрейме. Надо только наловчиться в правой панели выделять свойство через ЛКМ. Потом выделить другой фрейм и просто через Ctrl + V вставить.

## Layout

- Это сетка, нужна для удобства разметки
- Существует внутри фрейма
- Включается так
  - Выбираем фрейм, в меню справа ищем пункт Layout Guide
  - Добавляем сетку
    - Есть три варианта - клетка, колонки, строки
    - Для сайтов обычно используются колонки
      - обычно 12 штук
      - Тут же можно задать маргин и расстояние между колонками
        - дефолты 100 и 20
- Переключение показать \ скрыть - `Shift + G`

# Текст

- хоткей `T`
- если блок текста большой, а текст внутри маленький, 2хЛКМ по границе блока схлопывает его до размеров текста
- Подчеркивание, заполнение (лево, центр, по ширине) все делается в правой панели, раздел Typography > кнопка Type Settings

## Фишки

### Масштабирование

- Чтобы не гадать, какой размер шрифта нужен, можно инструментом scale можно растянуть блок текста до нужного размера
- Потом посмотреть, какой размер шрифта получился и выставить до ровного значения, чтобы не было дробей
- При этом надо поменять инструмент на Move, потому что иначе в правой панели не будет свойства размера шрифта
  - Вообще содержимое этой панели во многом, похоже, зависит от выбранного инструмента

### Обрезка текста ...

- Если текста много, и хочется сделать, чтобы он заканчивался на ... автоматически
  - Выбираем блок текста
  - В правом меню раздел Typography
  - Кнопка Type Settings
  - Пункт Truncate text > Truncation enabled
  - Задаем, сколько строк текста хотим видеть

# Auto-layout

- хоткей `Shift + A`
  - или в контекстном меню есть такой же пункт
  - `Ctrl + Shift + G` - TODO
- TODO: с виду что-то вроде объединения нескольких элементов в один контейнер
  - но загуглить получше, как именно оно работает
- Внутри auto-layout элемент можно перемещать вверх и вниз стрелками
- Можно задать расстояние между элементами, в правом меню искать Gap
- Если задать для layout скругление, а внутри например картинка, и ее края не скруглены
  - то надо для layout в правой панели найти Clip content галочку
- Выравнивание
  - Концептуально auto-layout похож на флекс, тут тоже можно равнять по лево \ право \ центр
    - Важно, можно Gap ставить в Auto, чтобы между элементами было автоматическое одинаковое расстояние
    - Направление элементов тоже можно ставить горизонтальное, вертикальное
      - Искать надо в правой панели в разделе Auto layout
      - Тут же как будто можно и Grid режим поставить
- Отвязка от лэйаута
  - Бывает засунешь элемент в блок, который является авто-лэйаутом, но этот элемент хочется позиционировать свободно
    - Т.е. он как бы часть a-l, но позиционирование хочется свободное
  - В этом случае можно его отвязать
    - Выбираем элемент и в правом меню в секции Position ищем такой квадратик Ignore Auto Layout
- практические кейсы

# Заливка и обводка

- Хоткеи
  - `Shift + X` - поменять местами заливку и обводку. Например, элемент был залит красным, а обводки не было. Нажимаем, теперь нет заливки, а есть красная обводка.
- Способы заливки
  - Сплошной \ градиент
  - Картинка!
    - Вот это интересно - картинка реализуется как способ заливки.
