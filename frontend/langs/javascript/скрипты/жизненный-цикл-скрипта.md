# Жизненный цикл скрипта

- Браузер
  - Скачивает скрипт
  - Парсит
    - Формирует AST (Abstract Syntax Tree) на основе исходного кода
    - Формирование шаблонов окружений
  - Компилирует
    - На основе AST получается байт-код
  - Выполняет
    - Байт-код выполняется
    - Переменным присваиваются значения
    - Выполняются функции
