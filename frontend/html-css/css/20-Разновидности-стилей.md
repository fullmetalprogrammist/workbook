# Разновидности стилей

## По способу подключения

- **Встроенные стили** (inline)
  - Пишутся непосредственно в теге через *атрибут* `style`
- **Внутренние стили** (internal, embedded)
  - Пишутся непосредственно в HTML-документе, в секции head через *тег* `style`
- **Внешние стили** (external)
  - Пишутся в отдельном css-файле и подключаются через тег `link` в секции head



## По "источнику" стилей

Возможные источники стилей:

- **Стиль браузера** (user agent style)
  - Стили, которые браузер по умолчанию применяет к элементам
    - Даже "не стилизованный" элемент должен как-то выглядеть визуально, поэтому у браузеров есть дефолтные стили для каждого элемента
    - У каждого браузера стили свои, вот почему например "не стилизованное" поле ввода может выглядеть по-разному в разных браузерах
- **Стиль пользователя** (user style)
  - Стили, которые пользователь настроил для страницы самостоятельно
    - Например, увеличил размер шрифта, изменил тип шрифта в настройках браузера
    - Или установил расширения, которые меняют стили страницы
      - Например, DarkReader, который может любую страницу перекрасить в темную тему
      - Блокировщики рекламы скрывают рекламу именно силами css
    - Или сам с помощью средств браузера составил какие-то стили
- **Стиль автора** (author style)
  - Стили, которые программист написал для страницы
  - Так сказать, это стили, которые должны показывать страницу так, как задумал ее автор
  - Загружаются вместе со страницей с сервера, кэшируются браузером
