# Каскад

- **Каскадом** называют очередность применения стилей
- По умолчанию очередность применения такая: стили браузера > стили пользователя > стили автора
  - Если в стилях есть пересекающиеся свойства, то конечное значение возьмется из того стиля, который применен последним
    - Т.е. стили применяются как бы "внахлёст" - что было позже перекрывает то, что было раньше
- Получается приоритет стилей такой:
  - Стиль автора - самый приоритетный
  - Стиль пользователя - средний приоритет
  - Стиль браузера - низкий приоритет
- Ключевое слово `!important` может изменить приоритет
  - Пользовательский стиль с important > авторского стиля без important
  - Авторский стиль с important > пользовательского с important или другого авторского без important
  - Если у двух "одинаковых" стилей (например, авторский и еще один авторский) есть important, тогда конечный результат определяется по **специфичности**

# Наследование

Наследование - это когда дочерние элементы перенимают значения свойств от своих родителей. Наследуемых свойств немного: большая часть из них относится к параметрам шрифта (шрифт, выравнивание, отступы) и цвета.
