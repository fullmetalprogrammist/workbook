- Классы моделей, которые сгенерировала призма, надо импортировать из `generater/prisma/client` файла

```typescript
import { Line } from "@/src/backend/generated/prisma/client";
```

- Еще есть вариация `-Model`, чтобы явно отличать призмовские модели

```typescript
import { CharacterModel, LineModel, SceneModel } from "@/src/backend/generated/prisma/models";
```
