# Как добраться до кода ошибки

- Через этот класс можно получить доступ к списку ошибок призмы

```typescript
import { Prisma } from '@/src/backend/generated/prisma/client';
```

- "Известные" ошибки имеют определенный тип + код

```typescript
if (error instanceof Prisma.PrismaClientKnownRequestError) {
  if (error.code === 'P2002') {  // Нарушение unique ограничения
    throw new Error('Код проекта уже занят');
  }
}
```
