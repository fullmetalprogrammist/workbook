- Установка

```
npm install react-hook-form zod @hookform/resolvers
```

- React Hook Form используется обычно в паре с zod
  - zod нужен для описания схемы данных формы, плюс позволяет задать правила валидации
- Обе библиотеки поддерживают TypeScript из коробки, дополнительных пакетов для этого не надо
