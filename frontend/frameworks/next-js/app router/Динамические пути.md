# Правила создания папок

- Допустим, хотим путь `projects/edit/dnd`, где `dnd` - динамическая часть
- Надо создать папки `projects/edit/[id]/page.tsx`
  - Там где хотим динамический сегмент, берем в квадратные скобки
  - id использовать не обязательно, можно например projectCode и т.д.
    - но возможно тогда придется перезапустить dev-сервер, без этого может не работать



# Извлечение в ручке динамической части

```typescript
interface EditProjectProps {
  params: Promise<{
    projectCode: string  // названние как у динамической части
  }>;
}

export default async function EditProject({ params }: EditProjectProps) {
  const { projectCode } = await params;  // извлекаем динамическую часть, next хранит ее в params
  console.log("Перешили на страницу редактирования проекта " + projectCode);

  return (
    <EditRoom />
  );
}
```
