# Что такое сеть докера

- Каждый контейнер по умолчанию запускается в отдельном net-неймспейсе
  - Это дает разные удобства
    - Например, порты не конфликтуют. Несколько контейнеров отлично могут слушать порт 80.
- "В одной сети" - не значит "в одном net-неймспейсе"
  - TODO: докер организует сеть по-другому как-то



# Запуск контейнеров в одном net-неймспейсе

- net-неймспейс нельзя создать самому
- Надо запустить какой-то контейнер, докер создаст ему неймспейс
  - Затем запустить второй контейнер, указав первый контейнер, и т.о. он запустится в том же net-неймспейсе

```
docker run --name foo fooImage
docker run --name bar --network container:foo barImage 
```
