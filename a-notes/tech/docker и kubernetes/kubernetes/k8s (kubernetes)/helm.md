- Helm — это пакетный менеджер для Kubernetes, который позволяет конфигурировать и устанавливать приложения в кластер.
- Helm Chart — это набор шаблонов для всех необходимых Kubernetes-манифестов (Deployment, Service, ConfigMap и т.д.), упакованный в единый архив.
  - Шаблоны - это не просто ямлы кубер-ресурсов с переменными вместо значений, а реально полноценные шаблоны с ветвлениями, циклами и т.д.
- Вендор упаковывает свое приложение в докер-образ (или несколько приложений в несколько образов), создает шаблоны.
- Мы составляем values-файл и устанавливаем с помощью helm приложение в свой кластер.
  - helm берет значения из values-файла, подставляет их в шаблоны и т.о. получаются готовые ямлы с ресурсами кубера
  - helm передает их куберу, он создает ресурсы и все - приложение установлено в кластер
